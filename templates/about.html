About

This website keeps track of fire engine inventories and equipment certification dates.

From my understanding of my local fire departments, accounts would be created by County IT and issued to relevant parties. For the purposes of this demonstration, I've only created one account in the users database. Simply log in using 'admin' and 'password' to access the system.

If a table row highlights when you hover over it, it is clickable. If you are using Microsoft Edge, it will not support the highlighting of rows, but they are still clickable.
